What is Oxygen OS

It is a pure (No bloatware), light OS written in rust designed to be as fast as possible while still maintaining key features. As well as that it is fully custom and doesnt relate to linux. This repository contains the source code, documentation, and community contributions.

Features

  • Community made
  • Optimized performance
  • Custok made kernel in Rust
  • Community-driven updates

Get Started

  1. Clone the repository: git clone https://github.com/Ruben2163/Oxygen-OS.git
  2. Follow the documentation in the README.md file.
  3. Contribute to the project by creating pull requests or reporting issues.
