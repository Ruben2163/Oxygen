What is LumOS?

LumOS is a pure, bloat-free, lightweight operating system written in Rust. Designed to deliver top-notch performance, it offers a customizable and efficient user experience while maintaining key features. Unlike traditional systems, it is fully custom and doesn't rely on Linux. Explore the repository for the source code, documentation, and community contributions.
