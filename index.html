What is Oxygen OS

Oxygen OS is a pure, lightweight operating system written in Rust, designed for optimal performance while maintaining essential features. It is fully custom and independent of Linux. This repository contains the source code, documentation, and community contributions.

Features

  • Community-driven development
  • Optimized performance
  • Custom-made kernel in Rust
  • Regular updates

Get Started

  1. Clone the repository: git clone https://github.com/Ruben2163/Oxygen-OS.git
  2. Follow the documentation in the README.md file.
  3. Contribute to the project by creating pull requests or reporting issues.
